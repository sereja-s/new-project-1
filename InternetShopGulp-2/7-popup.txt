1- в файле app.js подключим:
		// ПОПАП
		allFunctions.initPopups();

2- в файле style.scss подключим:
		// стили попап (модальных окон)
		@import "base/popup";
также укажем:
		body {
	// Скролл заблокирован
	.lock & {
		overflow: hidden;
		touch-action: none;
	}
}
3- раскомментируем попап в файле: src/html/popup.html и укажем:
		id="callback" // идентификатор попапа   +  в src/html/header.html пропишем в ссылке открывающей попап: href="#callback" и добавим класс: popup-link

		для закрывающей кнопки (ссылки) укажем: 
			href="#header" // ссылаемся на фиксированную шапку с id="header" (чтобы при закрытии модального окна, не возвращаться на начало страницы)  +  в src/html/header.html пропишем для header: id="header" и добавим класс: lock-padding
