1- в FIGMA -> перед тем как сохранить, выделяем иконку -> заходим в меню -> Object -> Outline stroke (обводка контура) или CTRL+SHIFT+O
2- переходим по адресу: https://icomoon.io/app/#/select -> Untitled Set(Набор без названия)-меню -> Import to Set -> выбираем все иконки в папке: iconsfont (сюда предварительно выгрузили все иконки проекта)
3- нажимаем на полоску с цветами и в открывшемся окне нажимаем на иконку корзины (удаляем цвета)
4- нажимаем на значёк: выбора (вверху) и потом протягиванием выбираем все наши иконки -> внизу нажимаем Generate Font
5- В сгенерированом шрифте (вверху) -> в preferences (настройки) можно увидеть(изменить) название шрифта и префикс его класса -> внизу нажимаем: Font (Download)
6- в папку: src/fonts сохраняем  из скаченного архива иконочного шрифта файл: icons.ttf Из файла style.css (из скаченного архива иконочного шрифта) нам нужны только названия классов иконок -> копируем их -> вставляем в наш файл ниже уже заготовленного кода со стилями для классов с иконками: src/scss/fonts/icons.scss
7- запускаем gulp в терминале командой npm run dev -> автоматически будет создан файл: src/scss/fonts/fonts.scss с подключенным в scss иконочным шрифтом
8- в нашем файле: src/scss/style.scss пропишем: @import "fonts/fonts"; и @import "fonts/icons";
